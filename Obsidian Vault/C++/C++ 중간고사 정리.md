### 복사생성자 호출되는 3가지 경우
1. 객체가 만들어지자마자 다른 객체로 초기화 될때
2. 객체가 함수의 매개변수로 전달될때
3. 객체가 함수로부터 반환될때
	- **임시객체가 만들어짐**으로써 복사생성자가 호출
---
```C++
int main()
{
	Point p1 ("p1", 10); //매개0 생성자 호출
}
```

Point p = p1; // 복사 생성자를 호출

// 임시객체 반환 후, **임시객체**가 소멸 -> 소멸자 호출 1
